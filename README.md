# SmartSpend
SmartSpend is a web-based financial dashboard that helps users manage expenses by extracting receipt data using OCR.

## Features
- User Login & Signup
- OCR Receipt Upload
- Expense Tracking
- Dashboard & Reports

## Tech Stack
- HTML
- CSS
- JavaScript
- Python
- Flask
- MySQL
